#Описание проекта
Проект представляет собой веб-форму для обратной связи, созданную с использованием Flask и HTML-шаблонов.

#Структура проекта
contact.html — основная форма обратной связи

base.html — базовый шаблон

static/ — статические файлы (CSS, JS)

templates/ — папка с HTML-шаблонами

#Основные компоненты
Форма включает следующие поля:

Имя пользователя

Email

Поле для сообщения

Кнопка отправки

#Функционал
Валидация ввода

Отображение сообщений об ошибках

Система уведомлений

#Установка
Клонируйте репозиторий:

bash
git clone https://github.com/ваш-репозиторий.git
Установите зависимости:

bash
pip install -r requirements.txt
Запустите приложение:

bash
python app.py
Использование
#Для работы с формой:

Заполните поля формы

Нажмите кнопку “Отправить”

Получите уведомление о результате

#Технологии
Python 3.x

Flask — веб-фреймворк

HTML/CSS — верстка

Git — система контроля версий
